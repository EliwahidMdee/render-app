## ⚡ Quick Start (3 Steps)

### 1️⃣ Install Dependencies
```bash
cd live_agent_app
flutter pub get
```

### 2️⃣ Configure API
Edit `lib/core/constants/api_constants.dart`:
```dart
static const String baseUrl = 'https://your-api-url.com';
```

### 3️⃣ Run the App
```bash
flutter run
```

**That's it!** 🎉 Your app is now running!
